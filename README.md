This repository contains the implementation of Binary Kummer Lines: BKL251 and BKL257, corresponding to the article **Binary Kummer Lines and Binary Edwards Curves: Efficient Two-way Vectorizations** published at *ACM ASIA CCS 2026*.

<h3>Authors of the codes:</h3>
<ol>
  <li><i>Sabyasachi Karati</i>,</li>
  <li><a href="https://scholar.google.com/citations?user=9etnbVwAAAAJ&hl=en" target="_blank"><i>Rittwik Hajra</i></a>, and</li>
  <li><i>Tanushri Sen.</i></li>
</ol>

<h3>Description:</h3>
<ol>
  <li><b>BKL251-SIMD-256</b>: This directory contains implementations of two-way vectorized fixed-base and variable-base scalar multiplications using BKL251. Each subdirectory corresponds to a specific polynomial multiplication strategy, as indicated by its name.
  </li>
  <li><b>BKL257-SIMD-256</b>: This directory contains implementations of two-way vectorized fixed-base and variable-base scalar multiplications using BKL257. Each subdirectory corresponds to a specific polynomial multiplication strategy, as indicated by its name.
  </li>
</ol>

Each subdirectory includes a <code>run.sh</code> script that provides the commands required to compile and execute the corresponding code.
